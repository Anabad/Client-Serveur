# Client-Serveur

## Introduction

Ce projet est un client-serveur de gestion d'une flotte de vehicules autonomes.
Le server a été créé en EmberJS et utilise la specicfication JSON-API.

## Installation
```
cd Serveur
npm install
cd ../Client
npm install
```
## Lancement du projet

```
cd ../Serveur
npm start
cd ../Client
npm start
```

Aller à l'url http://localhost:8080
